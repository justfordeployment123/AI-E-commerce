# Scraper Improvement Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace broken BackMarket/MusicMagpie Playwright scrapers with Jina.ai HTTP reader, add Envirofone as a fourth source, fix CeX laptop matching, and fix run management (deduplication + stuck run cleanup).

**Architecture:** The scraper service (`apps/scraper`, port 3003) uses Playwright only for CeX (Algolia API interception). BackMarket, MusicMagpie, and Envirofone switch to plain `fetch` via `r.jina.ai/{url}` which renders pages server-side and returns clean markdown — bypassing Cloudflare. The API (`apps/api`, port 3002) proxy-layer (`scraper-data` module) gains a `/scraper/cleanup` endpoint. Admin UI gains an Envirofone column and a stuck-runs cleanup button.

**Tech Stack:** NestJS (scraper + API), Prisma + PostgreSQL, Playwright (CeX only), Jina.ai Reader (free, no API key), Next.js (admin UI)

```typescript
// ─── Jina.ai Reader ───────────────────────────────────────────────────────────
// Fetches a URL via r.jina.ai which renders the page server-side and returns
// clean plain text — bypasses Cloudflare without needing a headless browser.
private async fetchWithJina(targetUrl: string): Promise<string> {
    const jinaUrl = `https://r.jina.ai/${targetUrl}`;
    const res = await fetch(jinaUrl, {
        headers: { 'Accept': 'text/plain', 'X-Timeout': '30' },
        signal: AbortSignal.timeout(35_000),
    });
    if (!res.ok) throw new Error(`Jina.ai returned HTTP ${res.status}`);
    return res.text();
}

// Extracts the first £NNN price from Jina.ai markdown output.
// Prefers a line that also contains the storage string (e.g. "128GB").
private extractPriceFromMarkdown(markdown: string, storage: string): number | null {
    const lines    = markdown.split('\n');
    const storageN = storage.toLowerCase().replace(/\s+/g, '');

    // Pass 1: prefer a line that mentions the storage variant
    if (storageN) {
        for (const line of lines) {
            if (!line.toLowerCase().replace(/\s+/g, '').includes(storageN)) continue;
            const m = line.match(/£\s*(\d+(?:\.\d{1,2})?)/);
            if (m) return parseFloat(m[1]);
        }
    }

    // Pass 2: first price on any line, ignoring trivial values like £1
    for (const line of lines) {
        const m = line.match(/£\s*(\d+(?:\.\d{1,2})?)/);
        if (m) {
            const val = parseFloat(m[1]);
            if (val > 1) return val;
        }
    }

    return null;
}
```

```typescript
private async scrapeBackMarket(query: string, storage: string): Promise<CompetitorPrices | null> {
```

```typescript
    this.logger.log('Starting competitor price scraper…');

    // Mark any stuck runs (RUNNING for > 2 hours) as FAILED
    const twoHoursAgo = new Date(Date.now() - 2 * 60 * 60 * 1000);
    const { count: stuckCount } = await this.prisma.scraperRun.updateMany({
        where: { status: 'RUNNING' as any, startedAt: { lt: twoHoursAgo } },
        data:  { status: 'FAILED' as any, finishedAt: new Date(), errorMessage: 'Run timed out — marked failed automatically' },
    });
    if (stuckCount > 0) this.logger.warn(`Marked ${stuckCount} stuck run(s) as FAILED.`);

    // Deduplication: skip if a run is already in progress
    const existing = await this.prisma.scraperRun.findFirst({
        where: { status: 'RUNNING' as any },
    });
    if (existing) {
        this.logger.warn(`Scraper already running (run ${existing.id}) — skipping.`);
        return {};
    }

    const run = await this.prisma.scraperRun.create({
        data: { status: 'RUNNING' as any },
    });
```
